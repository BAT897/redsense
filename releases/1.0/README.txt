- This build includes Oracle and SQLServer support
- OpenNLP model included for Portuguese language
- Dictionaries are not included
- Basic commands are:

1) java -jar RedataSense.jar --help
2) java -jar RedataSense.jar database-discovery -c => Column Discovery
3) java -jar RedataSense.jar database-discovery -d => Data Discovery
4) java -jar RedataSense.jar file-discovery

Refer https://github.com/redglue/redsense for more details
