DataAnonymizer
==============

Prerequisites
-------------
1. JDK 1.7+
2. Maven 3+

How to build DataAnonymizer
---------------------------
1. Download ZIP file
2. Unzip in directory of your choice
3. cd DataAnonymizer-master
4. cd DataAnonymizer
5. mvn install
6. DataAnonymizer.jar will be located in "target" directory
